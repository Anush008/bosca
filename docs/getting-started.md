Getting Started
===

Required Development Software
--
* Docker or Podman
* Docker Compose
* Go 1.22.2
* Node 20
* PNPM (https://pnpm.io/)

Setup Development Environment
--
* Clone https://github.com/sowers-io/bosca
* Run `./scripts/initialize-dev`

Start Services
--
* Run `./scripts/start-services`
